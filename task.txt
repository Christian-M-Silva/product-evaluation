- Iniciar aplicação com CRA com ts

- Criar interface de produto chamado IProduct
    - Nome: string
    - valor: number
    - avaliação: 1,2,3,4,5
    - descrption: string

- Criar um card component estiliza com taiwldin
    - Ele espera receber props do tipo IProduct
    - O nome do produto a esquerda e o valor na direita
    - O card por padrão só aparece essas informações, ao clicar nele, deverá fechar todos os outros cards e expandir ele com as descrições, clicando de novo fecha ele


- Usando redux irei armazenar as actions e os dados aqui
    - O valor armazenando é do tipo IProduct[]
    - Ele iniciará o array contendo valores imaginarios, ai aqui incluo os meus produtos só para simular

- No arquivo APP criar um .map usando o card component para renderizar os produtos puxando os valores do redux
